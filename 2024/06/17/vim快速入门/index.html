vim快速入门

vim快速入门

快捷键

模式

正常模式：无需操作/[Esc]
命令模式：[?] [/]
编辑模式：[i]
可视模式：[v]字符可视
         [V]行可视
         [ctrl+v]块可视

注：可视模式通俗的讲就是选中，选中范围为：高亮部分+光标所在位置（可能没高亮）。也就是说只要进入了可视模式，至少会被选中一个字符

光标移动

[h]左
[j]下（像向下的箭头）
[k]上
[l]右
[w]下一个单词开头
[b]上一个单词开头

复制剪切粘贴

[x]单个字母删除
[d]删除
[dd]删除当前行
[]
[y]复制
[yy]复制当前行
[p]粘贴

撤销重做

[u]撤销
[ctrl+r]重做

组合技

[ggVG]全选      [gg]到最顶行首[V]行可视[G]到最末行首

功能

[: set nu]显示行号
