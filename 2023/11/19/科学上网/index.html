概念简介(非专业)

市面上卖的VPN一般都是定制的客户端，即买即用。
而通常情况下，我们并不会想花钱买，由此就会去薅免费机场，比如：

几鸡-学习助理(下面会以这个为例)

有了机场，我们就需要借用代理连接到机场，常见的有:

Clash for Windows(官方已挂，不提供官方链接这个链接可以将就一下)
Clash for Android(官方已挂，不提供官方链接这个链接可以将就一下)
ShadowsocksR (for Windows)
ShadowsocksR (for Android)

机场注册

刚刚说了，以几鸡为例，so，请打开几鸡，注册一个账号，这个过程就不需要我演示了吧。

Q:为什么要用几鸡？
A:几鸡是一个部分免费，稳定的机场。每日签到可获得7天账号有效期、几G的流量。月初恢复10G.

注册好后完成签到。

windows 教程

下载ShadowsocksR (for Windows)
解压后打开ShadowsocksR-dotnet2.0.exe（ShadowsocksR-dotnet4.0.exe可能会报错）(个人感觉没有比Clash For Windows好用)

如果有安全询问请关闭

打开软件后如果有弹出软件的UI界面请先关闭，如果没反应就不管。

看看任务栏是否多了一个小飞机图标（默认白色）

右键任务栏小飞机图标，点击选择服务器目录下的SSR服务器订阅设置…，在弹出的窗口内点击Add，然后在新建的一个配置内输入
