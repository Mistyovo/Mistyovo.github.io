下载和安装

下载Visual Studio Code代码编辑器

点击这里转到Visual Studio Code官网

下载对应版本的安装包（或压缩包），安装过程略

下载C++编译器

主流的C++编译器有MinGW和mingw-w64可供选择，推荐使用后者，官网网址

点击上方Download按钮

点击Mingw-W64-builds点击Sourceforge跳转到一个国外的网盘

点击Files（上面的步骤太麻烦了？->便民链接）

滚动条往下拉，可以选择版本

版本 使用
MinGW-W64 Online Installer 在线安装版，不建议在国内下载，速度贼慢
MinGW-W64 GCC-版本号 整合包，为了方便下载，这边直接提供了8.1.0的便民下载链接（快夸我快夸我）
版本 版本
x86_64-posix-sjlj(推荐) x86_64-posix-seh
x86_64-win32-sjlj x86_64-win32-seh
i686-posix-sjlj i686-posix-dwarf
i686-win32-sjlj i686-win32-dwarf

下好后解压到如下位置（建议，因为我之后都会拿这个举例子）：

C:\Program Files\mingw-w64\
(为啥是这个路径呢？因为在线安装版的默认位置就是这)

复制bin文件夹的路径：

C:\Program Files\mingw-w64\x86_64-8.1.0-release-posix-sjlj-rt_v6-rev0\mingw64\bin

回到桌面，右键“计算机”，点击“属性”，点击“高级系统设置”，点击“环境变量”

找到PATH双击，没有就“新建”

变量名 变量值
PATH (上面复制的路径)

然后“确定”->“确定”->“确定”

按键盘上的Win+R，输入cmd回车

输入g++回车，如果出现：

g++: fatal error: no input files
compilation terminated.

则说明配置成功了，输入g++ -v可查看版本信息
