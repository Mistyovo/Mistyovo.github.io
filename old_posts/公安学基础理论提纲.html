公安学基础理论提纲

第一章 警察与公安

第一节 警察与警察学

1.警察的中文词源

  1. 中国固有说
  2. 日本舶来说
  3. 折中说

2.警察的概念

国家按照统治阶级的意志，运用武装的、行政的、刑事的手段，以强制性实力维护国家安全与社会治安秩序的行为

警察的本质是阶级性

警察的手段具有强制性

警察的目的是维护国家安全与社会治安秩序

第二节 警察的起源与发展阶段

1.警察起源观

  1. 警察自然起源观：警察与人类共生，是一种超阶级、超国家的自然现象，并把警察与国家看成永恒的现象
  2. 马克思主义警察起源观：警察与国家同步产生，是指人类社会进入奴隶社会之际，随着国家的产生，警察才成为必要

2.古代警察时期

  1. 警察产生的历史条件

    （1）经济条件：生产力的发展，私有制的产生

    （2）阶级条件：阶级矛盾以及统治阶级内部矛盾的不可调和性

    （3）社会条件：维护统治秩序与惩罚犯罪的客观需要

    （4）政治条件：国家机器的形成

  2. 古代警察的共同特点

    （1）警察行为尚未能集中于一个统一的专门机关，而是由军队、审判机关和行政机关共同行使

    （2）警察行为在法律上是不严格的，皇帝、“神灵”、各级行政长官意志对警察行为具有主导作用

    （3）私刑普遍存在，奴隶主、地主、宗教首领、宗族头人有权使用私刑执行惩罚

3.近代警察时期

  1. 产生的条件

    （1）经济条件：由于资本主义生产社会化的高度发展，市场经济的建立，社会财富在城市地区的空前积累，对社会治安秩序稳定的需求远胜从前

    （2）阶级条件：无产阶级反对资产阶级的斗争不断高涨，资产阶级需要加强对内的镇压力量，维护统治

    （3）社会条件：资源分配不平衡导致各类犯罪骤增，治安成为全社会的焦点问题

    （4）政治条件：资产阶级的人权思想和法制观念、平等要求，排斥古代的私刑制度，要求人身强制统一的由国家的警察力量依法施行

  2. 特征

    （1）警察行政的职能独立化

    （2）形成多层次的警察组织及专门工作系统

    （3）强调法制

    （4）统一制式服装

4.警察与国家

  1. 警察必须与国家一致

    （1）警察必须与国体一致

    （2）警察必须与国家意志一致

  2. 警察是国家机器的重要组成部分

  3. 警察是有实力的国家力量

    （1）暴力性实力

    （2）行政强制性实力

    （3）刑事侦查实力

    （4）防护性实力

  4. 警察的基本职能由国家的基本职能决定

第三节 公安与公安学

1.公安的概念

广义：是指社会秩序、公共财产、公民权利等社会各个方面的安全状态，以及包括社会成员为了公共安全而做的一切相关工作。

狭义：是指在中国共产党领导下形成的人民警察维护国家安全与社会治安秩序的安全状态，也指我国公安机关和公安民警的简称。

2.公安学的概念

公安学，是所有公安学科的总称，是中国共产党领导下的公安工作实践经验的总结和概括基础上形成的关于我国公安工作规律和对策的知识体系。

第四节 公安学基础理论

1.公安学基础理论的概念

公安学基础理论，是对我国公安实践经验的总体概括，是阐述公安工作一般规律和基本对策的理论体系，它既是对我国公安工作长期历史经验的总结，也是对我国各项公安工作共同规律、经验的总结。

2.公安学基础理论的指导思想

马克思列宁主义、毛泽东思想、邓小平理论、“三个代表”重要思想、科学发展观、习近平新时代中国特色社会主义思想，特别是习近平法治思想和人民民主专政的理论，是公安学最重要、最直接的指导思想

3.公安学基础理论的研究对象

公安学基础理论的研究对象，是在中国共产党领导下的公安现象中的基本关系，着重研究公安工作的一般规律与基本对策。

4.公安学基础理论的研究意义

  1. 是公安学一级学科建设和完善的需要
  2. 将公安学诸二级学科联结为有机整体
  3. 为公安学研究和公安工作提供理论工具
  4. 对公安学研究和公安工作起到指导作用
  5. 有助于树立科学的人民警察观

第三章 公安机关的性质和宗旨

第一节 公安机关的性质

1.公安机关的性质

公安机关的性质，是人民民主专政的重要工具，是具有武装性质的国家治安行政和具有刑事司法属性的专门机构。

公安机关是人民民主专政的重要工具（政治属性）

公安机关是具有武装性质的国家治安行政和具有刑事司法属性的专门机构（组织属性）

与我国其他国家行政机关有明显的区别

公安机关的政治属性表明了它的根本属性：

  1. 公安机关性质与我国国体一致
  2. 公安机关在我国国家政权中占有重要地位
  3. 公安机关是国家意志忠实执行者

公安机关的组织属性的具体体现

  1. 公安机关是国家治安行政机关

  2. 公安机关具有刑事司法属性

    （1）公安机关具有刑事司法职能

    （2）公安机关在刑事司法活动中的地位重要

  3. 公安机关具有武装性

    （1）军事性

    （2）暴力性

    （3）强制性

2.公安机关的政治属性和组织属性的要求

  1. 政治属性的要求

    （1）公安机关必须毫不动摇地坚持“公安姓党”

    （2）公安机关必须坚持政治建警

    （3）公安机关坚持站在广大人民群众的政治立场上

    （4）公安机关必须坚持廉政勤政，防止队伍蜕化变质

  2. 组织属性的要求

    （1）防止在公安工作中引入商品交易的关系

    （2）合理设置、改革公安组织机构

    （3）严格公安队伍管理

    （4）坚持谨慎用权、依法用权

第二节 公安机关的宗旨

全心全意为人民服务是中国共产党的根本宗旨，也是公安机关的根本宗旨

1.公安机关宗旨的内涵

  1. 服务对象是“人民”
  2. 服务的出发点是一切为了人民
  3. 服务态度体现为“真诚”“完全”“彻底”
  4. 服务内容既包括为人民群众排忧解难，也包括严格执法活动

第四章 公安机关的职能和任务

第一节 公安机关职能

职能，是指一定的人员、事务和组织机构所具有的功能和应起的作用

警察职能，是指警察机关或警察自身所具有的功能以及在其职责范围内应发挥出来的维护社会成员利益的效能和作用

1.公安机关职能的含义

公安机关职能是公安机关对国家和社会所应发挥的效能与作用（，体现着国家赋予公安机关的使命和要求）

2.公安机关职能的分类

  1. 政治关系角度（基本职能）

（1）专政职能

（2）民主职能

  1. 公安机关任务角度

（1）打击犯罪职能

（2）维护治安职能

（3）行政管理职能

（4）服务人民职能

  1. 法律角度

（1）治安行政管理职能

（2）刑事司法职能

（3）社会服务职能

  1. 社会转型角度

（1）执法职能

（2）管理职能

（3）服务职能

3.公安机关的基本职能

  1. 公安机关的专政职能

    概念：是指公安机关依法对危害国家安全、蓄意破坏和推翻社会主义制度以及严重危害社会治安秩序和人民利益的敌对分子和敌对势力进行镇压、制裁、控制和改造，以巩固人民民主专政，保障人民权利的社会效能

  2. 公安机关的民主职能

    概念：是指公安机关坚持“全过程人民民主”，充分体现“一切为了人民”“一切依靠人民”，即服从人民的意志，全心全意为人民服务，依靠人民当家作主的主人翁精神，调动人民的治安积极性，处理好人民内部矛盾的社会效能

    主要内容：

    （1）服务人民

    （2）保护人民

    （3）依靠人民

    （4）正确处理人民内部矛盾

  3. 公安机关专政职能和民主职能的联系

    从两种职能的联系上看，专政职能与民主职能不仅是相互依存、缺一不可的，而且是相互渗透、互相促进的，联系极为紧密，不可割裂或偏废

第二节 公安机关的任务

1.公安机关的任务的概念

我国公安机关的任务，是指公安机关在国家法律所确定的管辖范围内，为实现一定的目标所进行的工作内容。

2.公安机关的基本任务

  1. 维护国家安全
  2. 维护社会治安秩序
  3. 保护公民的人身安全、人身自由和合法财产
  4. 保护公共财产
  5. 预防、制止和惩治违法犯罪活动

第六章 公安机关职责和权力

第一节 公安机关职责

1.公安机关职责的概念

公安机关职责，是指国家依法确定的公安机关的管辖范围和责任义务。它是依据《人民警察法》确定的人民警察的职务责任，是由人民警察的性质与任务决定的，表明人民警察“应该”做什么，“必须”做什么

2.公安机关职责的特性

  1. 法定性
  2. 有限性
  3. 强制性
  4. 发展性

第二节 公安机关权力

1.警察权的概念

广义：是指国家有关警察活动的一切权力，它是国家权力体系的重要组成部分，包括国家关于警察工作的立法权、行政权和司法权

狭义：是指法律赋予警察机关及其警察人员实行警务活动时享有的权力。

我们通常所说的警察权是指狭义的警察权

2.公安机关权力的概念

公安机关权力，是指公安机关在履行职责过程中依法可以采取的权威性措施和手段

3.公安机关权力的特点

  1. 法定性
  2. 强制性
  3. 特许性
  4. 单向性
  5. 有限性

第七章 公安工作系统及特点

第一节 公安工作概述

  2. 狭义：是我国人民民主专政政权工作的重要组成成分，是公安机关人民警察在党和政府的领导下，根据法律和法规所从事的维护国家安全和社会治安秩序的专门工作

2.公安工作的含义

  1. 公安工作是我国人民民主政权工作的重要组成部分

  2. 公安工作是依据国家法律和法规保卫国家安全和维护社会治安秩序的专门工作

第二节公安工作系统

  4. 隐蔽性与公开性相结合
  5. 集中型与分散性相结合

第八章 公安工作的根本原则

党对公安工作的绝对领导，是我国公安工作的根本原则

第一节 党对公安工作的绝对领导

1.党对公安工作绝对领导的含义

  1. 在公安机关的归属问题上，在我国的各种政治力量中，中国共产党对公安工作具有绝对的领导权，由中国共产党而不是其他任何政治力量可以和能够领导公安工作的
  2. 在党与公安机关的关系上，必须明确是中国共产党缔造了公安机关，公安工作是党的工作不可分割的组成部分。

2.党对公安工作领导的内容

  1. 政治上的领导
  2. 思想上的领导
  3. 组织上的领导
  4. 法制上的领导
  5. 决策上的领导

3.党对公安工作领导的特点

  1. 独立性
  2. 无条件性
  3. 全面性
  4. 直接性

第二节 公安工作必须坚持党的绝对领导

1.公安工作坚持党的绝对领导的必要性

  1. 公安机关是人民民主专政的重要工具，只有在党的领导下才能维护国家政治安全、社会稳定和人民安宁
  2. 公安机关的武装性特点及拥有的强制性权力，决定了其必须在党的领导之下
  3. 公安工作要处理社会矛盾中最具有对抗性、隐蔽性和腐蚀性的问题，在党的领导下才能保证公安机关的战斗力和纯洁性
  4. 公安工作具有广泛的社会性，需要党的领导去动员、组织和协调各方面的力量
  5. 党的领导是正确制定公安决策的根本保证

2. 公安工作坚持党的绝对领导的主要表现

  1. 政治立场坚定
  2. 思想观念牢固
  3. 组织队伍可靠
  4. 法律意识过硬

3.公安工作必须置于党委的实际领导之下

  1. 自觉接受党委的领导

  2. 大力加强各级公安机关党的建设

    （1）大量加强公安机关领导班子建设，这是坚持党对公安工作领导的关键环节

    （2）抓好公安机关基层党支部的建设，充分发挥战斗堡垒作用

    （3）加强对党员的教育和管理

  3. 严禁把侦察手段用于党内

第九章 公安工作的根本路线和基本方针

第一节 公安工作的根本路线

1.群众，人民、人民群众的概念

  1. 群众：泛指人民大众；指没有加入共产党、共青团组织的人；不担任领导职务的人
  2. 人民：泛指社会的全体成员
  3. 人民群众：是指一切对社会历史起着推动作用的人们

2.公安工作群众路线的含义

公安工作群众路线，是公安工作实行的一切为了群众，一切依靠群众，从群众中来，到群众中去所遵循的根本途径。它是公安工作宣传群众、依靠群众、服务群众、保护群众、组织群众的理论、原则、制度和方法的总称

一切为了群众，是公安工作的出发点和归宿，是公安工作的宗旨和职责的有机结合

一切依靠群众，是公安工作的根本态度

从群众中来，到群众中去，是公安工作的基本方法

3.公安工作坚持群众路线的必要性

  1. 公安机关的内在属性所决定
  2. 公安机关的宗旨要求所决定
  3. 公安工作的客观需要所决定

4.公安工作坚持群众路线的基本要求

  1. 树立群众观念
  2. 提高工作本领
  3. 接受群众监督

第二节 公安工作基本方针

1.公安工作基本方针的概念

公安工作的基本方针，就是党委领导下专门机关与广大群众相结合。

专门机关与广大群众相结合的内涵，就是在党委的领导下，把公安机关的职能作用与人民群众的积极主动精神结合起来

2.坚持公安工作基本方针的途径

  1. 搭建警民沟通渠道
  2. 落实便民利民措施
  3. 动员组织人民群众
  4. 开展群防群治工作

2.落实公安工作基本方针的现实意义

  1. 促进社会和谐稳定
  2. 推进社会治理创新
  3. 构建和谐警民关系
  4. 建立基层多元参与治安治理机制
